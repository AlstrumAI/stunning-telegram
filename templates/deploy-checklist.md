# Weekly Deploy Checklist

## Friday (COT Day)
- [ ] COT data drops at 3:30pm ET
- [ ] Relay updates on alstrum.ai and app.alstrum.ai
- [ ] Screenshot the Relay from app
- [ ] Post Instagram story: Relay screenshot + Substack link sticker
- [ ] Tweet: "[Biggest positioning shift]. Full breakdown in Monday's Breaking Structure. Relay live at alstrum.ai"
- [ ] Note top 3 positioning shifts for Saturday's writing

## Saturday (Writing Day)
- [ ] Write edition. Polish. Add charts/screenshots.
- [ ] Schedule on Substack for Monday morning (6am or 9am ET)
- [ ] Paste final markdown into `published/current-edition.md`
- [ ] Update `data/metrics.md` with last week's numbers
- [ ] Update `data/audience-signals.md` if readers asked notable questions
- [ ] Git commit + push

## Sunday (Batch Generation)
- [ ] Pull repo on whatever machine you're using
- [ ] Open CC session. Say: "Generate this week's distribution from current edition."
- [ ] CC reads CLAUDE.md → context/ → current edition → templates
- [ ] CC generates: angles.md, all platform posts, all video data JSONs
- [ ] Review every piece. Tweak voice where needed.
- [ ] On Windows box: render each video composition via render.sh
- [ ] Git commit staged content + push

## Monday (Newsletter Day)
- [ ] Newsletter auto-sends from Substack
- [ ] Post monday-twitter.md content via Typefully
- [ ] Post monday-linkedin.md (link in first comment)
- [ ] Post monday-insta-carousel (via Canva or direct)

## Tuesday-Thursday
- [ ] Deploy remaining staged content daily
- [ ] 15-20 min engagement on target accounts (see strategy/engagement-targets.md)

## Friday
- [ ] Loop restarts. COT drops. Relay updates. Screenshot. Tweet.

## Minimum Viable Week (Emergency Mode)
- [ ] Newsletter publishes (already scheduled Saturday)
- [ ] Friday Relay screenshot story on Instagram
- [ ] One tweet with the strongest data point
