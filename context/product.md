# Product Context

## Stryk — The Platform
URL: app.alstrum.ai
The operating system of modern trading. AI layer of capital markets.

Four Progressive Modes:
1. Show Me — see what institutional money is doing
2. Guide Me — actionable signals with plain-English explanations
3. Do It For Me — semi-automated within user rules
4. Full Autopilot — fully automated ML-driven trading

ML Architecture (AlphaNet): LightGBM + GBDT + Transformer + meta-learner ensemble. Multi-horizon predictions (5min to 3day). SHAP explainability on every signal.

Brokerage: SnapTrade (equities/options/ETFs), Oanda (forex).

## The Relay — The Growth Asset
Persistent capital flow visualization: Bonds → Currencies → Commodities → Equities → Crypto

Updated weekly when COT drops (Friday 3:30pm ET). Each node: contract count, net spec percentile, directional arrow, status badge, momentum score.

Lives at: alstrum.ai (public), app.alstrum.ai (interactive in-app).

The Relay is a top-of-funnel acquisition tool. Every Friday screenshot = free content. "See this week's Relay: alstrum.ai" drives traffic and return visits.

## Attribution Engine
14+ SHAP feature families → plain-English context. Powers signal cards and trade reasoning. Not a black box.
